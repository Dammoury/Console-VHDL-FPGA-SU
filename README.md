# Console-VHDL-FPGA-SU
Conception d'une console de jeux sur FPGA (Basys/Nexys) en VHDL dans le cadre du module 3ème année de Licence à Sorbonne Université
Ce dépôt contient le code VHDL de la console SU-EE100.
Points clés : implémentation d'un contrôleur VGA 4 bits, gestion d'un encodeur rotatif par machine à états, et logique de jeu (Casse-Briques).
